## V1 — 已完成
单 lane、纯几何白膜。核心对推机制（圣水回涨 / 8 张循环卡组 / 单位沿 lane 推进与碰撞 / 三塔制胜负）+ 3 种技能积木（生成 / 直伤 / AOE）+ 简单进攻型规则 AI。逻辑层与显示层彻底分离，纯逻辑层全单元测试覆盖，编辑器内可玩。
> **Step 9 安卓打包缓做**（编辑器即可体验/继续开发，需要分发到手机时再补）。

## V2 — 已完成
顺序 **A → D → B → C**：① **A** 3-lane 玩法深度 → ② **D** 美术/动画/音频换皮 → ③ **B** AI 攻防与难度分级 → ④ **C** 扩内容与数值平衡。**不做**空中/地面克制（留 V3）。细节见 [PLAN_V2.md](PLAN_V2.md)。

## V3 — 进行中
**从「技术验证物」做成「买断制轻中度单机产品」**。头号工程 = **战斗核心 2D 重构**：3-lane 简化丢失了 CR「兵自由走位的轻 RTS」乐趣，重构为 **2D 场地**（河 + 左右双桥 + 己方半场自由落点 + 流场绕桥寻路 + 完整仇恨/分心 + 软推挤碰撞 + 塔反击，取代 lane）。其后：**空地克制**（飞兵）、**新技能积木**、**混合主轴**（短战役引导 + Roguelite 终盘）、**2D 卡通精灵换皮**、交互手感、音频、**安卓+桌面导出**（原 V1 Step 9）、上架打磨。细节见 [PLAN_V3.md](PLAN_V3.md)。

## V4+ — 远期（若成真正产品）
- 联机对战、匹配、聊天、防作弊。
- 逻辑层已为「搬到服务器」留门：逻辑不依赖渲染、不绑帧率、玩家与 AI 都只向逻辑层发指令。

## 贯穿所有阶段的设计铁律（不变）
- **逻辑层 / 显示层彻底分离**；**玩家与 AI 对称**（都只向逻辑层发指令）。
- lane 位置用 `0.0~1.0` 进度，不用屏幕像素；**固定逻辑 tick（10Hz）**结算，不绑渲染帧，显示层做插值。
- 碰撞用「队列前后关系」纯逻辑判断，**不用物理引擎**。
- 数值 / 卡牌 **全走 JSON 配置**。
- **不过度设计**；一步一确认 + 每步 commit + 逻辑层步骤必配单元测试。
