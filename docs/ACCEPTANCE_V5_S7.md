## 前置
1. docker 服务起着：`cd server && docker compose up -d`（5 容器 pg/redis/gateway/api/battle）。
2. Godot 编辑器 **F5 跑主场景**（main_menu）。**Output 面板**会刷 `[V5]` 流程日志，辅助定位。
3. 想看服务端日志：`cd server && docker compose logs -f api`（请求行 + economy 业务行）。
4. 想重置存档/排序偏好：删 `user://`（编辑器 → 项目 → 打开用户数据文件夹）下的 `player_save.json` / `settings.cfg`（非必须）。

## 用例 1 — 基地 Base Camp（S7b）
- **入口**：主菜单 → `开始`。
- **操作**：进入基地，观察各元素。
- **预期**：① 顶部钱包条（金币/宝石，数值=Output `[V5][econ] 拉状态` 的 gold/gems）；② 「队伍战力」大字，**按下一关推荐着色**（≥推荐=绿 / 略低=金 / 偏低=红）；③ 挂机收益卡（待领>0 才显「领取」）；④ 金色「闯关·第N章」CTA；⑤ 养成/卡组/天梯 + 返回。
- **判定**：✅ 元素齐全、数值与服务器一致、按钮可跳转。

## 用例 2 — 闯关地图（S7c）
- **入口**：基地 → `闯关`。
- **操作**：看列表，滚动。
- **预期**：标题「闯关」+「下一关 …」；章节头「第1章 · 星 x/6」；关卡行三态——**当前关**金边高亮+「挑战」、**已通关**绿边+星级+「已通关」、**未解锁**暗+「锁定」+「通关前一关解锁」。
- **判定**：✅ 三态样式正确；当前关可点、锁定关不可点。

## 用例 3（重点：手感）— 闯关全流程 + 领奖开箱
- **入口**：基地 → 闯关 → 点当前关「挑战」→ 组卡界面点「出战」→ 战斗。
- **操作**：**打赢这一关**（推掉敌方公主塔/王塔，或限时内塔血占优）。胜利后点「返回」。
- **预期**：① 回到**闯关地图**（★关键：**不能**弹去「肉鸽征程」）；② 弹「领奖开箱」动画：宝箱抖动 → 弹开金光 → 星级逐颗点亮 → 金币/宝石/碎片滚动 → 「继续」；**点击可跳过到末态**；③ 关闭后地图刷新：该关→已通关+星级、下一关解锁、钱包增加。
- **判定**：✅ 战后回闯关地图（非肉鸽）；开箱动画播放且可跳过；星级合理（通关1星+保王塔/限时各+1）；奖励/进度/货币与服务端一致（对 Output `[V5][econ] 上报通关 … ok → gold=…`）。
- **失败分支**：打输（0星）→ 无开箱、无奖励、进度不变（Output `发奖结果 <null>`）。

## 用例 4（重点：KAN-67）— 养成卡格多维排序
- **入口**：基地 → `养成`。
- **操作 + 预期**（逐项点）：
  - a) **默认**：顶部分段控件 `[稀有度][费][等级][可养成]` + `[升序/降序]`；默认「稀有度」金色高亮、方向「升序」；卡格普通→传说排列。
  - b) 点「**费**」→ 卡格**即时重排**为圣水费升序（便宜在前）；「费」高亮。
  - c) 点「**等级**」→ 按等级·阶重排（默认降序，高的在前）。
  - d) 点「**可养成**」→ 可升级/可解锁的卡（带金色红点）**置顶**。
  - e) 点「**升序/降序**」→ 当前键方向翻转、即时重排，按钮文字切换。
  - f) **退出养成再进**（返回基地→再进养成）或**重开游戏**→ **记住**上次选的排序键+方向。
- **判定**：✅ 每次切换**即时重排**且顺序符合该键；高亮跟随选中键；方向翻转生效；重进记住选择。

## 用例 5 — 养成详情 升级/升阶/解锁（S7d）
- **入口**：养成 → 点任一**已解锁**卡。
- **操作**：① 点「升级」（金够时）；② 若到本阶等级上限→「升级」变灰提示「需升阶解锁」；③ 点「升阶」（碎片+金够时）；④ 金/碎不足时点→红字 toast「货币不足」、不扣。
  另：点一张**未解锁**卡 → 显示碎片进度条 + 「解锁」（够）/「再集 N 碎片」（不够）。
- **预期**：成功后数值条/等级/阶 pip 即时更新、钱包减少（对 Output `[V5][econ] upgrade/rank_up/unlock … ok → gold=…`）。
- **判定**：✅ 升级/升阶/解锁正确扣费并更新；满级/满阶/不足的拦截 + toast 正确。

## 用例 6 — deck builder 已解锁池 + 战力着色（S7e）
- **入口**：基地 → 闯关 → 挑战某关 → 组卡（或 基地 → 卡组）。
- **操作**：看卡池、增减卡。
- **预期**：卡池**只列已解锁卡**（不含锁定卡）；顶部「战力 X / 推荐 Y」**按达标着色**（绿/金/红）；选满 8 张「出战/保存」才可点。
- **判定**：✅ 池=已解锁；战力实时随选卡变化并正确着色；从基地「卡组」进=「保存」回基地（不进战斗）。

## 用例 7（回归）— stale-run 不再串味（修过的 bug）
- **入口**：主菜单 → `肉鸽闯关`（进去）→ `菜单`（返回，制造残留 run）→ `开始` → 闯关 → 挑战 1-1 → 出战。
- **预期**：Output 显示 `[V5][battle] 模式=闯关`（**不是** `模式=肉鸽`）；打赢后回**闯关地图**、正常发奖推进。
- **判定**：✅ 模式=闯关；战后回闯关地图（非肉鸽征程）。
