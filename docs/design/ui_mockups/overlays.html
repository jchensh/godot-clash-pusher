规范要点

  • 引导层压暗 72% + 挖洞软边 8~12px 羽化（程序遮罩）；手指两帧点击动效；军师半身像（Q 版三头身，P2 可后补，先纯气泡）；气泡带朝向尾巴两版
  • 设置由全屏页改为弹窗（导航简化）；开关/分段控件/列表行全部组件库拼装——本屏是组件库完备性的验收用例
  • 确认框560 小档；「取消左、确认右」，危险操作确认用红色按钮；有后果的确认框不给右上 ✕
  • Toast屏幕上 1/4 悬浮、2 秒自消、不响应点击、同屏仅一条；文案带小图标可选
  • 断线层战斗内=全屏压暗+中央面板（sim 冻结）；非战斗页=顶部横条不打断浏览；恢复后自动消失

给美术的资源点

  • 手指图标两帧、气泡框（尾巴左/右两版 9-slice）、军师半身像（可选 P2）
  • 开关（on/off）、断线图标；其余全复用窗口框/按钮/Toast 底
