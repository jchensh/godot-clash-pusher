结构说明

  • 顶栏高 104px 锚安全区下：我方名片（蓝底）+ 三王冠槽 | 中央倒计时（≤30 秒红色脉动+放大）| 敌方名片（红底，PVP 显示对手真名/头像，PVE 显示关卡 Boss 名）
  • 战场中区全部让给游戏画面；HUD 任何元素不得侵入战场纵向范围（部署、观察不被遮挡是铁律）
  • 圣水条10 格分段 + 当前数值 + 满槽发光脉动；紫色系与费用药丸同族
  • 手牌4 张（140×172 档）+ 左侧下一张预览（小号半透）；选中卡金描边上浮 10px；费用不足的卡整体压暗+费用数字变红
  • 认输白旗小圆钮挂战场右上角落（PVE=退出，PVP=认输二次确认）

给美术的资源点

  • 顶栏底条、名片框蓝/红两版、王冠槽（亮/暗）、倒计时数字美术字
  • 圣水条组件（底/格填充/满槽光）、圣水滴图标、手牌卡框四稀有度+选中态、下一张小框
  • 伤害跳字数字美术字三色（白/暴击黄/治疗绿）——飘在战场层但属 UI 资源
