结构说明（与 CR 截屏对位）

  • 1货币行：金币/宝石药丸 + 「+」商城位（未开先留）↔ CR 顶部货币行
  • 2名片横幅：头像(武将立绘) + 昵称 + 杯数；右侧小钮=公告/设置 ↔ CR 玩家横幅+功能钮
  • 3活动轨：左=挂机金库(带计时)，右=探险占位(灰) ↔ CR 左右活动图标轨；后续活动/礼包都挂这里
  • 4章节主视觉：当前章节场景图+闯关总进度 ↔ CR 竞技场展示+奖杯路进度，「随进度换景」是留给美术的表现空间
  • 5操作大簇：卡组(角标=张数) | 大对战按钮(主 CTA，天梯) | 闯关(角标=当前关) ↔ CR 卡组/对战/奖杯之路三件套
  • 6页签导航：商店(灰) 卡牌 对战(中央放大) 闯关 探险(灰)；对战页即本屏 ↔ CR 五页签，支持左右滑切页

硬规矩

  • 红虚线=安全区：顶 100px / 底 60px（详见《UI 系统策划案》第二章）
  • 全部按钮需四态；页签切换带滑动动效；探险/商店灰态但占位
  • 旧主菜单「六按钮列表」废弃，导航职责下沉到页签+大簇
