结构说明

  • 形态弹窗（非全屏页）：卡牌页压暗 60% 上弹出中窗口框；右上 ✕ 关闭（无副作用操作允许 ✕）
  • 大卡320×400 档 + 稀有度框 + 立绘（程序放置）；名称 + 称号短句 + 势力角标（魏蓝/蜀绿/吴红/群雄黄）
  • 等级与阶Lv + 阶 pip（金=已达，灰=未达，最多 4 pip）；升阶解锁新技能积木时在 pip 旁加说明行
  • 数值行攻/血/速度/射程 四行，升级预览增量绿字「+N ↑」；速度/射程为档位文字不是数字
  • 按钮态升级可负担=金色可点；资源不足=禁用态+成本数字变红；点升级播放数值滚动+按钮脉冲
  • 未解锁卡同弹窗但大卡为剪影+锁，按钮区变「解锁条件说明」（碎片进度/章节奖励来源）
