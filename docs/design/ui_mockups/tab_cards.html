结构说明

  • 合并旧「养成图鉴」「组卡」两页合并成本页：上=出战卡组（PVE/PVP 共用槽1），下=收藏池；点收藏卡与卡组互换，无需再进独立组卡页
  • 卡组区2×4 八张 + 平均费用 + 卡组战力；战力对比下一关推荐值着绿/琥珀
  • 排序四键分段控件（稀有度/费用/等级/可养成）+ 升降序按钮，记忆上次选择
  • 卡格三态已解锁（等级条）/ 可升级（右上红点）/ 未解锁（灰+锁+碎片进度条青色）
  • 稀有度边框四色：灰银/蓝/紫/金，传说带外发光

给美术的资源点

  • 卡格框 96×120 档四稀有度 × 三态；面板底、分段控件、排序方向钮
  • 卡面立绘为程序放置（用单位立绘），美术只出框
  • 长按弹出卡详情（见 card_detail.html）
