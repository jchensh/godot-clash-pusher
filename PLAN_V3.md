> 本文件是 **V3 阶段的权威规划**。V1（单 lane 白膜 demo）、V2（3-lane + 程序化换皮 + AI 难度 + 内容/平衡）已完成，历史规格见 [PLAN_V1.md](PLAN_V1.md) / [PLAN_V2.md](PLAN_V2.md)；全局 roadmap 见 [PLAN_GRAND.md](PLAN_GRAND.md)。
> **开发纪律沿用 V1/V2**：一步一确认、每步 commit、逻辑层步骤必配单元测试、数值/卡牌走 JSON、逻辑/显示彻底分离、玩家与 AI 对称、不过度设计。表现/手感/美术 类步骤交真人实机验收。

## 1. V3 目标一句话
把 V2 的「3-lane 简化原型」**重构成皇室战争式的 2D 轻 RTS 战斗**（自由落点 + 单位在场地上自己绕桥寻路、被仇恨吸引转火 + 塔会反击），再以**混合主轴（短战役引导 + Roguelite 终盘）** + **2D 卡通精灵换皮** + 交互手感 + 音频 + 上架打磨，做成**买断制轻中度单机**。

**头号工程 = 战斗核心 reboot（V3-1）**：当前「3 条离散 lane + 单位锁死 lane + 1D 队列碰撞」丢掉了 CR 的灵魂（兵自由走位的轻 RTS 乐趣）。V3-1 把**逻辑层空间模型从 1D lane 进度重构为 2D 场地**：河 + 双桥地形、己方半场任意落点、流场寻路绕桥、完整仇恨/分心、软推挤碰撞、塔反击。

## 4. 程序重构设计（V3-1 施工依据）

**原则不变**：逻辑/显示分离、固定 10Hz tick、玩家 AI 对称、**确定性无随机、可 headless 单测、不用物理引擎**。变的只是空间表示：1D → 2D。

### 4.1 坐标与场地
- **tile 空间**：场地 `W×H` 个 tile（草案 18×30，竖屏，可调），单位位置为连续 `Vector2`∈`[0,W]×[0,H]`，**非屏幕像素**；view 负责 tile→屏幕映射。
- **方向约定**：`y=0` 敌方底线（上），`y=H` 玩家底线（下）；河在中部。玩家兵朝 `y` 减小推进、敌方朝增大（对称）。
- **地形网格**：每 tile 类型 `GROUND / WATER / TOWER`。河 = 中部若干行 WATER，**左右两段桥**为 GROUND 缺口；塔占位 = TOWER（阻挡）。**空军忽略地形**（V3-2）。
- **塔布局**：每方 2 公主（前排左/右）+ 1 王（后排中），固定 tile 位 + 占位；写进新配置 `config/arena.json`（网格尺寸/河行/桥列段/塔位），`levels.json` 引用。

### 4.2 寻路（流场，确定性、廉价）
- 地形 + 塔静态 → 建图时对**每座塔**跑一次 BFS 距离场 `dist_B[tile]`（4 邻、地面可走）；只在**塔被摧毁**（占位释放、路径变）时重算对应场。
- 地面兵锁定建筑 B 时，朝 `dist_B` 最小的相邻 tile 走 → **自动绕到最近的桥过河**。锁定敌方单位时直线趋向（在 `aggro_radius` 内、短程，地形影响小）。
- **每 tick 每兵 O(1) 查表**，零随机。

### 4.3 目标获取 + 仇恨（完整 CR）
- 每兵持 `current_target`（单位或建筑）。重选时机：目标失效 / 周期 N tick。
- 规则：`aggro_radius` 内有敌方单位 → 取最近者（分心/被拉扯）；否则取最近敌塔（按流场/欧氏）。**确定性 tie-break**（spawn 序/id）。
- 预留 `targets_only_buildings`（巨人/气球式只攻建筑）→ 列待细化，首版可不开。

### 4.4 移动 + 软推挤碰撞
- 每 tick：算朝向（流场梯度或趋向目标）→ 移动 `move_speed*dt`；进 `attack_range` 则停下攻击。地面兵钳制在可走 tile（不进水/塔）。
- **软分离**：兵有 `body_radius`；移动后**固定顺序**遍历 pair，重叠则沿连心线各推开半个重叠量，迭代 1–2 趟；推后再钳出水/塔。固定顺序 = 确定。n≤~40，O(n²) 可接受（必要时后续加空间哈希）。
- 接敌、攻击、伤害结算、清死 沿用现有「收集后统一结算」模式（含塔攻击）。

### 4.5 改动波及面（文件级）
| 文件 | 处置 |
|---|---|
| `logic/lane.gd` | **删除**，由新 `logic/arena.gd` 取代（网格/地形/流场/全单位/tick：重选目标→移动→分离→攻击→清死，塔攻击也在此） |
| `logic/unit.gd` | `progress`→`pos:Vector2` + `velocity/body_radius/aggro_radius/current_target`；`attack_range` 改 2D 距离；去 `lane_index` |

### 4.6 可测性（headless 单测举例）
- 寻路：左下角出地面兵 → T tick 内 x 逼近左桥、越过河行（y 穿过河带）。
- 仇恨：敌方单位进 `aggro_radius` → `current_target` 切到它；移出/死亡 → 回锁塔。
- 分离：两兵初始重叠 → 数 tick 后间距 ≥ 半径和。
- 塔反击：敌兵进塔射程 → 按冷却掉血、最终被塔清。
- 落点：水/塔/越半场 → `try_play_card` 拒绝。
